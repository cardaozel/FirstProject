# First Project

iOS app built with Swift and Xcode. My first iOS project.

## Requirements

- Xcode 15+
- iOS 15.0+

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/cardaozel/FirstProject.git
   cd FirstProject
   ```

2. Open in Xcode:
   ```bash
   open FirstProject.xcodeproj
   ```

3. Run on simulator or device (⌘R)
